📖 설정 가이드 보기
1. script.google.com 접속
2. 새 프로젝트 클릭
3. 제공된 apps-script.gs 코드 전체 붙여넣기
4. 💾 저장 → 배포 → 새 배포
5. 유형: 웹 앱 선택
6. 실행: 나, 액세스: 모든 사용자
7. 배포 → 권한 허용 → URL 복사
8. 위 입력창에 붙여넣고 연결 저장
